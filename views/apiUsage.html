/user/<ID>

Returns information about user with <ID>, like name, address as JSON. Requires coworker privileges

{ prename: <prename>, name: <name>, points: <loyal points>, street: <street nr>,
city: <city>, additionalAddress: <additionalAddress>, postcode: <postcode>, accessLevel: <accessLevel> }

/account/

    .../login

      Send credentials via POST as { email: "<mail>", password: <password> }. If succedd it returns

      { prename: <prename>, name: <name>,
      points: <loyal points>, accessLevel: <accessLevel> }

      erro code 401 if credentials are wrong.


      Access Levels:

      Guest:0
      User: 5
      Coworker: 8
      Admin: 10

    .../logout

      Just send a POST request with "connect.sid" cookie set. The server will delete the associated session. This cookie can be deleted. On error just delete the "connect.sid" cookie anyway.

/news/

    .../new

      Just send a GET request and you will receive
      { title: <title>, message: <message>}

    .../all

      Just send a GET request and you will receive
      { news: <news[][]>}
