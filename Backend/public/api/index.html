/account/
    Diese Schnittstelle ist für das Anlegen sowie das Löschen von Nutzern zuständig.
    Weiterhin kann man sich hier anmelden, abmelden, Accountinformationen setzen und prüfen ob der Session-cookie angemeldet ist.

/cart/
    Mit dieser Schnittstelle kann man Gegenstäne in den Warenkorb hinzufügen, entfernen, Anzahl aktualisieren, auslese und bestellen.

/item/
    Diese Schnittstelle liefert alle Gegenstäne zurück

/news/
    über diese Schnittstelle kann man Neuigkeiten anlegen, auslesen, bearbeiten und auch löschen.

/user/
    Diese Schnittstelle liefert Informationen über einen bestimmten Nutzer
